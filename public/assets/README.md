# Game assets (SweetSprint)

Obstacle images are generated by `npm run generate:obstacles` and loaded from `assets/obstacle/`. You can replace them with custom PNGs.

## Expected paths (relative to `public/`)

| Path | Description | Suggested size |
|------|-------------|----------------|
| `assets/character/run.png` | Player run cycle (optional; generate via `npm run generate:spritesheet`) | 384×64 (6×64×64) |
| `assets/cookie.png` | Cookie collectible | 56×56 |
| `assets/obstacle/obstacle_vehicle.png` | Car obstacle | 120×70 |
| `assets/obstacle/obstacle_pet.png` | Dog obstacle | 56×50 |
| `assets/obstacle/obstacle_person.png` | Pedestrian obstacle | 40×90 |
| `assets/obstacle/obstacle_waterPuddle.png` | Puddle obstacle | 80×28 |
| `assets/cloud.png` | Cloud (parallax) | 110×50 |
| `assets/parallax_far.png` | Far background layer | 800×220 |
| `assets/parallax_mid.png` | Mid background layer | 800×220 |
| `assets/parallax_near.png` | Near background layer | 800×220 |

If a file is missing, the game uses its built-in illustrated procedural sprites. All assets use bottom-center or center origin where applicable.
