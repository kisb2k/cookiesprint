# **App Name**: SweetSprint

## Core Features:

- Runner Character Control: Enable user input for the girl to jump, slide, and move between three lanes to avoid obstacles.
- Procedural Lane Generation: Dynamically generate an endless stream of street lanes with varying widths and layouts as the player progresses, ensuring replayability.
- Cookie Collection Mechanic: Implement logic for the character to collect cookies spread across the lanes and update the player's score accordingly.
- Obstacle Interaction System: Detect collisions with various street obstacles (vehicles, pets, people, water puddles) and trigger appropriate game-ending conditions.
- Score and Progress Tracking: Display the player's current score (based on distance run and collected cookies) and provide visual indicators of game progression during gameplay.
- Dynamic Obstacle Placement Tool: Utilize AI to generate challenging and varied sequences of obstacles in real-time across the lanes, dynamically deciding on placement and type to create diverse and engaging gameplay patterns.

## Style Guidelines:

- Primary color: A warm, inviting coral (#DC634A) to evoke sweetness and active energy.
- Background color: A very light, desaturated peach (#F8F1EF) providing a bright and clean canvas for the street environment.
- Accent color: A vibrant pink (#ED5DA7) to highlight interactive elements, scores, and important UI feedback, creating visual contrast.
- Headline font: 'Poppins' (sans-serif) for a modern, contemporary, and precise game feel. Body/UI text font: 'Inter' (sans-serif) for legibility and a neutral, objective display of scores and instructions.
- Utilize playful, cartoonish icons for game elements, obstacles, and power-ups, consistent with the lighthearted game theme. Consider subtle animations for collected items like cookies.
- Implement a clean and focused layout where the game canvas dominates the screen. Essential UI elements like score and progress will be minimally overlaid, ensuring an immersive experience.
- Incorporate smooth and responsive animations for character actions (jump, slide), lane changes, obstacle appearance, and cookie collection feedback (e.g., sparkles or points tally). Provide a clear 'game over' animation sequence.
